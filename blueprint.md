# Project Blueprint: LinguaHub - Advanced Grammar for Adult Learners

## Overview
LinguaHub is an interactive language learning platform designed for adults. It provides practical, example-driven grammar explanations that help adult students immediately apply what they learn to real-life situations.

## Implemented Features
1.  **Multi-Language Support**: Korean, English, Chinese, Japanese, and **Spanish**.
2.  **Practical Grammar (Essential)**: A new category for each language featuring context-heavy examples and practical tips.
3.  **Modern UI/UX**:
    *   Web Components based architecture.
    *   Glassmorphism and interactive glow effects.
    *   Responsive design for mobile and desktop.
    *   "상세 문법" (Detailed Grammar) tab for deep learning.

## Next Steps: Deployment
1.  **Git Initialization**: Set up the local repository.
2.  **GitHub Prep**: Instructions for the user to push to GitHub.

## Technical Details
*   **Web Components**: `LanguageCard` for selection, `LessonHub` for content management.
*   **CSS Architecture**: `@layer` based styling with modern color spaces (`oklch`).
*   **Data Model**: Centralized `languageData` object in `main.js`.
